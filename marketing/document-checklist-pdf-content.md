### INTRODUCTION

Getting your visa approved starts with having the RIGHT documents, properly prepared. This checklist covers:
- ✅ Documents for ALL visa types
- ✅ Country-specific requirements
- ✅ Formatting guidelines
- ✅ Common rejection triggers
- ✅ Digital backup strategy

**BEFORE YOU START:**
- Start collecting documents 3-6 months before application
- Scan everything in color
- Keep originals AND copies
- Check expiry dates on all documents

---

## SECTION 1: UNIVERSAL DOCUMENTS (All Visa Types)

### Identity Documents
- [ ] Valid Passport (6+ months validity beyond travel date)
- [ ] Passport Photos (check country-specific size requirements)
- [ ] National ID Card
- [ ] Birth Certificate (with translation if needed)

### Application Documents
- [ ] Completed visa application form (online or paper)
- [ ] Application fee payment receipt
- [ ] Visa application appointment confirmation

### Travel Documents
- [ ] Detailed travel itinerary
- [ ] Return flight reservation
- [ ] Accommodation bookings (hotel Airbnb, host letter)
- [ ] Travel insurance certificate

---

## SECTION 2: FINANCIAL DOCUMENTS (Critical!)

### Bank Statements
- [ ] Personal bank statements (last 3-6 months)
- [ ] Salary/business income proof
- [ ] Tax returns (last 2 years if applicable)

**⚠️ CRITICAL RULE:**
Funds must show CONSISTENT balance for 28-90 days (depending on country). NO last-minute lump deposits!

### Sponsorship Documents (if applicable)
- [ ] Sponsor's bank statements (3-6 months)
- [ ] Sponsor's employment letter
- [ ] Sponsor's ID/passport copy
- [ ] Affidavit of support (notarized)
- [ ] Relationship proof (birth cert, marriage cert)

**💡 TIP:** Sponsors should be IMMEDIATE family (parents, siblings, spouse). Distant relatives = red flag.

---

## SECTION 3: PURPOSE-SPECIFIC DOCUMENTS

### STUDENT VISAS
**Educational Documents:**
- [ ] Admission/Acceptance letter from university
- [ ] CAS number (UK) / I-20 form (US)
- [ ] Academic transcripts & certificates
- [ ] English test results (IELTS, TOEFL, Duolingo)
- [ ] Statement of Purpose (SOP)
- [ ] CV/Resume

**Financial:**
- [ ] Proof of tuition payment (or funds for full first year)
- [ ] Living expenses proof ($10,000-$15,000 depending on country)
- [ ] Scholarship/grant letters (if applicable)

### WORK VISAS
**Employment Documents:**
- [ ] Job offer letter/employment contract
- [ ] Company registration (if self-employed)
- [ ] Qualifications/degree certificates
- [ ] Professional certifications
- [ ] Reference letters from past employers

### TOURIST VISAS
**Purpose Proof:**
- [ ] Tour itinerary
- [ ] Event invitations (weddings, conferences)
- [ ] Friends/family invitation letter
- [ ] Home ties evidence (job, property, family)

### BUSINESS VISAS
**Business Documents:**
- [ ] Invitation letter from host company
- [ ] Your company registration documents
- [ ] Business bank statements (6 months)
- [ ] Letter from your employer explaining trip purpose
- [ ] Conference/event registration

---

## SECTION 4: COMMON REJECTION TRIGGERS (AVOID!)

**🚫 Financial Red Flags:**
- Large deposits just before application
- Unexplained transactions
- Insufficient funds
- Wrong bank account type (use current/savings, not domiciliary for proof)

**🚫 Document Issues:**
- Missing translations (for non-English documents)
- Expired documents
- Blurry scans
- Incomplete applications

**🚫 Sponsor Problems:**
- Distant relatives as sponsors
- Sponsor can't be reached by phone
- Sponsor's finances don't match claimed support

---

## SECTION 5: DOCUMENT FORMATTING GUIDE

**Scanning Requirements:**
- Color scans (not black & white)
- Minimum 300 DPI
- PDF format preferred
- File size: Under 2MB per document

**Translation Requirements:**
- Official translator or certified translation
- Include translator's certification
- Original document + translation both required

**Organizing Digital Files:**
```
/VisaApplication
  /Identity
    - Passport.pdf
    - Photo.jpg
  /Financial
    - BankStatements.pdf
    - SponsorLetter.pdf
  /Education
    - AdmissionLetter.pdf
    - Transcripts.pdf
  /Travel
    - FlightBooking.pdf
    - Insurance.pdf
```
